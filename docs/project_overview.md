# Project Overview: AOE (Ask Open WebUI Everywhere)

## Description
AOE is a browser extension designed to integrate Open WebUI directly into the browsing experience via the side panel. It allows users to interact with AI alongside any web page without switching tabs.

## Key Features
- **Side Panel Integration**: Open WebUI instance in the browser's side panel.
- **Insert Text**: Parses current webpage content and attaches it as a .txt file to the chat.
- **Insert Screen**: Captures a screenshot of the visible tab and pastes it into the chat.

## Tech Stack
- **Platform**: Web Extension (Manifest V3)
- **Languages**: JavaScript, HTML, CSS
- **Target Browsers**: Chrome, Edge (Chromium-based)

## Configuration
Users can configure:
- **UI URL**: The address of the Open WebUI instance.
- **Target Match URL**: Pattern for permissions.
